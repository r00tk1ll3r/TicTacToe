# TicTacToe
#### Modified Version of  pkill-aditya/class12project

You can play again and again same game i mean in pkill-aditya/class12project if you select multiplayer after game over you have to go through all of the menu stuff but in my upgrade i created a loop so after game over you can play again without entering menu also i upgraded some record file

## Build

### Linux

just run `make` in project folder and that file create tictactoe executable to play

### Windows
Run `g++ Application.cpp Game.cpp Board.cpp -o tictactoe` on cmd
